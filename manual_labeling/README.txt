How to use manual labeling features

1. Open PPT, open desired slide
2. Run main_gui.py
3. Enter the name of the test, click on "start labeling"
4. This starts the listener. You cna now start selecting / labeling groups
5. Click finish when done
6. You'll automatically move on to text labeling (i.e. identifying chunks of text in textboxes)
7. If no text to label, click "fINISH'
8. Ignore any errors
9. Open the file generated, add manual labels you may have missed , save with test #, and move to after_processing folder
10. Run python file to convert to tree structure
